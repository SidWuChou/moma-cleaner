## рҹҡҖ Quick Start

### Installation

```bash
# Clone the repo
git clone https://github.com/SidWuChou/moma-cleaner.git
cd moma-cleaner

# Install dependencies
pip install -r requirements.txt

# Run!
python moma_cleaner.py --path ~/Downloads
```

### Usage

```bash
# Organize a folder
python moma_cleaner.py --path ~/Downloads

# Preview changes (don't actually move files)
python moma_cleaner.py --path ~/Downloads --dry-run

# Use AI to name files
python moma_cleaner.py --path ~/Downloads --ai-name

# Remove duplicates
python moma_cleaner.py --path ~/Downloads --dedup

# Schedule daily cleaning
python moma_cleaner.py --schedule daily
```
